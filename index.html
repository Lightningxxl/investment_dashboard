1投资目标与约束

这一组输入相当于投资政策：定义收益目标、风险预算、beta 暴露和组合集中度。

Required return%

组合预期年化收益下限。产品 μ = 历史年化 × 收益置信折扣（45%-100%），组合 μ 为持仓加权。低于下限判定"收益不足"。

Max drawdown %

组合最大回撤预算。组合回撤 = max(年度路径回撤, 加权单品回撤×相关性折扣)。点 i 查看完整估计方法。

Max standard deviation%

组合波动上限。组合 σ = √(wᵀ·Σ·w)，Σ 由单品波动率和两两相关性构成。Excel 明细优先，PDF 用年度收益标准差兜底。

Beta benchmark

所有产品的 Beta 和相关性均相对此基准计算。样本不足时用年度收益回归 + 策略先验做贝叶斯混合。切换后触发全量重算。

Max beta exposure

相对当前 Beta benchmark 的组合 beta 上限。样本不足时引入策略 beta 先验。

分析流程与模型口径

六步链路：每个步骤的输出是下一步的输入。点击顶部"方法说明"可查看完整推导和公式。

1. 投资政策 定义组合的四个硬约束。收益、回撤、波动和 Beta 必须同时满足，组合才进入可行集合。 组合收益 >= Required return 组合回撤 <= Max drawdown 组合波动 <= Max stdev 组合 beta <= Max beta
2. 参数质量 历史样本长度、低大盘相关性、低集中度风险、数据口径质量四个维度分别评分（0-100），加权平均得到参数质量分。该分数决定收益置信折扣（45%+55%×质量分/100）和单品权重上限。 参数质量分 = Σ(维度分 × 重要性) / Σ(重要性) 收益置信折扣 = 45% + 55% × 参数质量分 / 100 预期收益 μ = 历史年化 × 收益置信折扣 单品权重上限 = f(参数质量分, 集中度模式)
3. 候选池构建 四层过滤：资格过滤（申购状态+数据可用性+最低质量）→ 硬门槛（质量≥45+样本≥1年+口径≥50）→ 策略分层（每策略代表+低相关策略保护+优先级补足）→ 权重兼容（等权模式下剔除 cap 不足的产品）。 全池 → 资格过滤 → 硬门槛 → 策略分层 → 相似去重 → 候选池 策略分层: 每策略代表 → 低相关底线 → 优先级补足 去重规则: 相关性>0.86 同策略/管理人, 单管理人≤2只
4. 参数估计 Excel 明细优先：收益、波动、回撤、Beta、规模、净值跨度直接来自成立以来指标。PDF 年度数据兜底：年度收益均值/标准差/历史回撤。Beta 和相关性在样本不足时采用贝叶斯混合，引入策略先验。 收益/波动/回撤: Excel 明细 > PDF 年度 > 兜底估计 Beta: Excel β 匹配基准时直接用, 否则年度回归+策略先验混合 相关性: 日/周频(需≥12点) > 年度收益 > 结构先验 回撤(组合): max(年度路径回撤, 加权单品回撤×相关性折扣)
5. 组合优化 等权模式：固定 K 只基金，每只 1/K，精确枚举 C(N,K) 个组合后约束过滤。质量倾斜模式：五个确定性起点 + 2400 次固定种子随机搜索，权重有质量偏向并受单品上限和策略上限约束。 等权: K 只基金 → 枚举 C(N,K) → 四约束过滤 → 效用最优 倾斜: 候选池 → 多起点搜索 → 权重修正 → 效用最优 效用分 = μ×1.7 + 质量×0.22 - σ×0.75 - DD×0.65 - β超限惩罚
6. 结果解释 始终展示三类组合：效用最优（主推荐，平衡收益/风险/质量）、最低风险（σ+DD 最小，保守方案）、收益上沿（μ 最大，激进方案）。无可分解时展示最接近目标的近似解。点击任意组合卡片查看完整推导详情。 ┌─ 效用最优: U 最大 — 主推荐 ├─ 最低风险: σ + DD 最小 — 保守边界 └─ 收益上沿: μ 最大 — 激进边界 固定种子 = 20260618 → 同参数同结果, 可复现可审计
